# Gateloom — public brand & documentation assets

Brand marks (SVG) and UI snapshots (PNG) for Gateloom, published here so they
can be embedded in external documentation (e.g. Notion). The Gateloom source
code lives in a separate, private repository.

- `brand/` — the Woven G logomark: light, dark, animated, and favicon variants.
- `snapshots/` — UI captures of the landing page, MUX lab, and brand sheet in
  light and dark themes.

These assets are © Gowri R Hiremath. Published for documentation embedding,
not for reuse.
